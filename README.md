# mp2_textanalysis
A Python-based text analysis app that examines how news articles frame events through language. Inspired by the concept of “Permission to Narrate,” this project compares word choices across sources to identify patterns in tone, bias, and negative connotation, with visualizations to support the analysis.

## Project Structure
functions/
